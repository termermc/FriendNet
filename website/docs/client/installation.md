# Client Installation

Head over to the [download](/download/) page to download the client for your platform.

The first time you run the program, it will ask you to install a certificate to your system.
That certificate is required to enable HTTPS on `localhost`, which is needed for the web UI
to work properly. Otherwise, you will see errors in your browser and the page will not show.

If you still see an error in your browser after installing it, restart your browser and open
it again.
